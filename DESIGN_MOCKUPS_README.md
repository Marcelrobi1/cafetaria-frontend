## 🚀 Cómo Visualizar

### Método 1: Abrir en Navegador
```bash
# Simplemente abre el archivo HTML
open DESIGN_MOCKUPS.html
# o en Windows
start DESIGN_MOCKUPS.html
```

### Método 2: Convertir a PDF
1. Abre `DESIGN_MOCKUPS.html` en Chrome o Edge
2. Presiona `Ctrl + P` (Windows) o `Cmd + P` (Mac)
3. Destino: "Guardar como PDF"
4. Marca: "Gráficos de fondo"
5. Guarda como: `Design_Mockups_Cafeteria.pdf`

### Método 3: Con Servidor Local
```bash
# Si tienes Python instalado
python -m http.server 8000

# Luego abre en el navegador:
# http://localhost:8000/DESIGN_MOCKUPS.html
```
