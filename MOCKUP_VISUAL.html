🎯 Descripción General

Sistema web completo para la gestión integral de una cafetería, que incluye funcionalidades tanto para clientes como para administradores. La aplicación permite visualizar el menú, realizar pedidos, gestionar ingredientes, platos, menús y usuarios.

🛠️ Stack Tecnológico

React 19 Vite React Router Context API ESLint JavaScript ES6+ CSS3 HTML5 LocalStorage API

👥 Funcionalidades para Clientes

  • Página de Inicio Atractiva: Hero section, sobre nosotros, productos destacados
  • Menú Digital Completo: Visualización de catálogo con imágenes y descripciones
  • Carrito de Compras: Agregar/eliminar productos, ajustar cantidades
  • Sistema de Autenticación: Registro e inicio de sesión seguro
  • Perfil de Usuario: Gestión de información personal
  • Experiencia Responsive: Adaptable a móviles, tablets y desktop

🔧 Panel de Administración

🥗 Gestión de Ingredientes

CRUD completo, control de inventario, precios

🍽️ Gestión de Platos

Crear, editar, eliminar platos con imágenes

📋 Gestión de Menús

Menús del día, combinaciones, precios especiales

💰 Gestión de Compras

👤 Gestión de Usuarios

🚀 Instalación y Ejecución

# Clonar el repositorio
git clone https://github.com/Marcelrobi1/cafetaria-frontend.git

# Instalar dependencias
npm install

# Ejecutar en modo desarrollo
npm run dev

# Compilar para producción
npm run build

# Vista previa de producción
npm run preview
